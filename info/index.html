Acasă Simple Structurate

Variabile

O variabilă reprezintă o locație de memorie unde se află o valoarea de un anumit tip, variabilele fiind caracterizate de o adresă, identificator, domeniul și tipul acesteia


Variabile Simple

Variabile care stocheaza doar o valoare

int varsta = 69;

Variabile Structurate

Variabile mai complexe care contin mai multe elemente, precum date folosind struct si matrici

int matrice[3][4];
