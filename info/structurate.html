Acasă Simple Structurate

Variabile Structurate

    Acest tip de variabile sunt variabilele care contin mai multe elemente.

  • Vectori

    • Vectorii sau tablourile unidimensionale sunt structuri de date bine definite si organizate in memorie. Cu ajutorul acestora, se pot păstra in memorie si accesa ulterior mai multe variabile, fără a fi nevoie de reținerea explicită a fiecăreia dintre ele.

      Pot crește sau micșora dinamic.

      Oferă funcții utile pentru manipularea datelor (inserare, ștergere, sortare etc.).

    Exemplu:

    #include 
using namespace std;
int v[100];
int main() {
 ...
 return 0;
}
  • Matrice

    • În C++, matricele reprezintă o structură de date standard, folosită pentru a stoca o colecție de date de același tip, organizate pe linii și coloane. Practic, o matrice este extinderea la două dimensiuni a unui vector, de unde și numele alternativ de tablou bidimensional.

    Declarare

    int A[3] [4]; //Asa declaram o matrice cu 3 linii si 4 coloane.

    Citire::

    cin >> n >> m;
for(int i = 0 ; i < n ; i ++)
    for(int j = 0 ; j < m ; j ++)
        cin >> A[i][j];};

    Afisare

    for(int i = 0 ; i < n ; i ++)
{
    for(int j = 0 ; j < m ; j ++)
        cout << A[i][j] << " ";
    cout << "\n";}
  • Struct

    • Struct este o structura de date care iti permite sa grupezi mai multe elemente sub o singura umbrela. Aceste elemente sunt denumite “membrii” si pot avea tipuri diferite. De exemplu pentru definirea unei structuri ce reprezinta o masina, analizam codul de mai jos:

    Declarare

    struct masina{
    char marca[50];
    char culoare[20];
    int capacitate_rezervor;
    int an_fabricatie;
    float pret;
    float km_parcursi;
} V[20]; // Definim un vector cu 20 de masin
  • Probleme matrici

    • Problema 1

    Se dă o matrice mat cu >= 1m >= 1 linii șin >= n >= 1 coloane, având toate elementele distincte. Să se interschimbe liniile pe care se află cel mai mic și respectiv cel mai mare element din matrice.

    Problema 2

    Se dă o matrice mat cum >= 1m >= 1 linii și n >= 1n >= 1 coloane. Să se rotească cu 90 de grade sensul acelor de ceasornic matricea dată.
  • Probleme vectori

    • Problema 1

    Se dă un șir cu n elemente, numere naturale. Determinați diferența în valoare absolută dintre numărul de valori pare din șir și numărul de valori impare din șir.

    Problema 2

    Se citește un vector cu n elemente, numere naturale. Să se înlocuiască ultimul element prim din vector cu 0, apoi să se afișeze vectorul.
  • Probleme struct

    • Problema 1

    Se citesc datele despre elevii unei clase, respectiv numele şi data naşterii. Să se ordoneze elevii în ordinea crescătoare după data naşterii şi să se afişeze această situaţie.

    Problema 2

    Să se afişeze reprezentarea internă a unei date de tip caracter folosind reprezentarea pe biţi.
